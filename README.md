# Q&A Система с гибридным поиском

Гибридная система поиска ответов на вопросы, объединяющая классический TF-IDF и современные эмбеддинги от Sentence Transformers.

## 🚀 Возможности

- **Гибридный поиск**: комбинация TF-IDF для точных терминов и эмбеддингов для семантического поиска
- **Кэширование эмбеддингов**: быстрый старт после первого запуска
- **Поддержка русского языка**: модель paraphrase-multilingual-MiniLM-L12-v2
- **Работа на CPU**: оптимизирована для Mac M1 и других CPU
